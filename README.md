## General info
This project is created to cover the service ["Продуктовый помощник"](https://foodgram-frontend-1.prakticum-team.ru/recipes) by UI tests.

## api directory
Includes api methods for related endpoints.

## pages directory
You can find all files with each Page Object in ['pages'](../blob/main/pages) directory.

## tests directory
You can find all files with the tests for specific page in ['tests'](../blob/main/tests) directory.

## pipeline screenshots directory
Created to store screenshots with successfully passed GitHub Actions pipeline 

## Additional files & directories
There are several auxiliary files and directories in the project

#### 1. helper
Created to store additional methods.

#### 2. urls
Created to store urls.

#### 3. requirements
Contains the list of packages or libraries needed to work on a project.

#### 4. data directory
Created to store test data.

#### 5. locators directory
Created to store files with locators for each page.

#### 6. Other additional directories and files
Additional files used for packaging tests into a Docker image, launching the environment using Docker Compose with Selenoid, and configure a CI/CD pipeline for automated project builds via GitHub Actions.
